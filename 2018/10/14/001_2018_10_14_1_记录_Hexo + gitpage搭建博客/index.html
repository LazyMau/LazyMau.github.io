Hexo + GitPage 搭建博客

下面有些代码是手打的，若错误，可复制搜索查找正确的。

参考资料：https://www.jianshu.com/p/f4cc5866946b

windows:

（输入的命令都是在cmd或者PowerShell下）

安装git

自行百度

设置用户名和邮箱

生成密钥

会在默认路径下会多出来.ssh/ 其中包含了私钥文件id_rsa和公钥文件id_rsa.pub

将id_rsa.pub里的内容复制粘贴到github->头像->setting->SSH and GPG keys->new SSH key 里面的key文本框，ssh名字自己随意

添加成功后验证：

成功后返回的字符串里会显示你的github的用户名

创建GitHub仓库

1）登陆自己github

2）以自己的用户名.github.io作为仓库名创建仓库

安装hexo

要安装好npm和node.js，参考：http://www.runoob.com/nodejs/nodejs-install-setup.html

创建博客文件夹

在本地找个文件夹 或者重新创建一个blog文件夹

安装hexo

npm install hexo-cli -g

初始化hexo

hexo init

安装依赖

hexo install

启动hexo

hexo server

启动之后就可以在http：//loaclhost:4000看到示例，ctrl+c可以停止运行

部署hexo

安装git插件

npm install hexo-deployer-git --save

部署

hexo deploy 或 hexo d

或者 hexo d -g

第一句和第二句是等价的，第三句等于 hexo g && hexo d

没问题就部署成功了，用https://username.github.io即可访问
